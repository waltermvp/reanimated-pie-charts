# react-native-reanimated-pie-charts

Pie Charts build using reanimated

## Installation

```sh
npm install react-native-reanimated-pie-charts
```

## Usage

```js
import { multiply } from 'react-native-reanimated-pie-charts';

// ...

const result = await multiply(3, 7);
```
